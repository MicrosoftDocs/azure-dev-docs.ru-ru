#### <a name="identify-application-performance-management-apm-agents"></a>Определение агентов управления производительностью приложений

Найдите все агенты наблюдения за производительностью приложений, которые используются в этих приложениях (например, Dynatrace или Datadog). Вместо этих агентов в Azure Spring Cloud Azure реализована глубокая интеграция с Azure Monitor для управления производительностью и реагирования на аберрации в режиме реального времени. Дополнительные сведения см. в разделе [После миграции](#post-migration).
