## <a name="clean-up-resources"></a>Очистка ресурсов

Чтобы очистить все ресурсы, используемые во время этого краткого руководства, удалите группу ресурсов:

```azurecli
az group delete \
    --name $AZ_RESOURCE_GROUP \
    --yes
```
